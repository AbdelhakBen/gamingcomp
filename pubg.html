RÈGLES DÉTAILLÉES

Dans le cas d’un retard d’un participant ou plusieurs, ce dernier aura une marge de 10 minutes maximum pour se présenter. Après le temps écoulé, la partie se lancera, et les retardataires seront disqualifiés.
Toute déconnexion pendant la partie engendra la disqualification de la personne.

La partie de PUBG Mobile aura la configuration suivante :
  • Map : elle sera annoncée au début du tournoi
  • Le reste des paramètres par défaut.

Dans la phase de qualification, les dix (10) premiers personnes avec le plus de points de chaque fin de partie, seront retenues pour la phase suivante.

Le système de points est le suivant :

  • Chaque Kill vaudra un point (1pt).
  • Dans le cas d’égalité de point, la personne avec le classement le plus élevé sera retenue.
