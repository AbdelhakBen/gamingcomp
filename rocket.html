RÈGLES DÉTAILLÉES

Dans le cas d’un retard d’un participant ou plusieurs, ce dernier aura une marge de 10 minutes au maximum pour se présenter :
  • Si la/les personne(s) se présente(nt), la partie pourra être lancée.
  • Si aucun membre de l'équipe n'est présent après le temps écoulé, l’équipe adverse gagnera automatiquement par forfait.
Note : Pour les personnes inscrites en équipe, Le capitaine peut remplacer l’absent(s) par remplaçant(s)

Dans chaque partie, l'organisateur résponsable choisira une équipe pour créer une partie privée avec:
  • Name : NomEquipe1_NomEquipe2
  • Password : au choix

La partie de Rocket League aura la configuration suivante :
  • CrossPlatform : Activé (Xbox one, Ps4, PC)
  • Game Mode : SOCCER
  • Serveurs : Europe
  • Stade : DFH Stadium
  • Mutator Settings : Series Length: 3 games and the rest are default settings.

Si un membre de l’équipe se déconnecte pendant la partie, la partie pourra continue sans lui.
